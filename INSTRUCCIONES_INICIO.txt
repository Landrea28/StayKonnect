INSTRUCCIONES DE EJECUCIÓN
===========================

CONFIGURACIÓN BASE DE DATOS
---------------------------
Antes de iniciar, edita el archivo:
BackEnd/gretta/src/main/resources/application.properties

Comenta o descomenta la sección correspondiente a tu usuario de base de datos.


POSTGRESQL (Windows)
--------------------
# Iniciar:
net start postgresql-x64-18

# Apagar:
net stop postgresql-x64-18


GUÍA RÁPIDA DE TERMINAL
-----------------------
cd <carpeta>   : Entrar a una carpeta (ej: cd BackEnd)
cd ..          : Regresar a la carpeta anterior
ls             : Listar archivos en la carpeta actual (en Windows usar 'dir')


--- BACKEND ---
1. Navegar a la carpeta del backend:
   cd StayKonnectApp/backend

2. Iniciar la aplicación Spring Boot:
   mvn spring-boot:run


--- FRONTEND ---
1. Navegar a la carpeta del frontend:
   cd StayKonnectApp/frontend

2. Instalar dependencias (si es la primera vez):
   npm install

3. Iniciar el servidor de desarrollo:
   npm start
